# Servidor principal para gestion de usuarios y sus respuestas


Para ejecutar por primera vez. 

Prerrequisitos.

Crear una base de datos mysql llamada "proyectoencuestas" o en su defecto el nombre que se indique en el archivo 
database.js

Esa base de datos debe tener 2 tablas relaciondas entre si usuarios y respuestas, utilizar como base la db.sql que se encuentra en la carpeta db, para la estructuracion de la base de datos.




Ubicarse en la carpeta mysqlApiNode

ejecutar los comandos 

npm install 

y 

npm start

Al iniciar dara un mensaje de montaje del servidor exitoso

server in the port  3000
conected to the database
